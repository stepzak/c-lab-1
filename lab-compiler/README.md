# Работа GCC

## Этап 1: Препроцессинг
    
### Используемые типы ошибок:
    
* Неизвестная директива
* Ошибочная директива(например, #include несуществующего файла)

## Этап 2: Компиляция

### Используемые типы ошибок:

* Неизвестная переменная(функция и т.д.)
* Неправильный тип переменной(функци и т.д.)

## Этап 3: Ассемблирование:

### Используемые типы ошибок:

* Неизвестная инструкция
* Неправильная инструкция(н., неправильный синтаксис известной инструкции)

## Этап 4: Линковка

### Используемые типы ошибок:

* Объявление без реализации
* Несколько разных реализаций одного и того же объявления


# Запуск

```sudo chmod +x run_errors.sh```

```./run_errors.sh```
